# Karabiner: переключение между двумя раскладками (игнорируя системную)
## Проблема
На macOS 3 раскладки: Australian (системная, нельзя удалить), RussianWin, m4de qwerty (кастомная). Нужно переключаться только между двумя нужными (RussianWin и m4de qwerty).
## Как узнать input_source_id раскладок
Для системных: `defaults read ~/Library/Preferences/com.apple.HIToolbox AppleEnabledInputSources`
Для кастомной: переключиться на неё, затем `defaults read ~/Library/Preferences/com.apple.HIToolbox AppleCurrentKeyboardLayoutInputSourceID`
Кастомные раскладки лежат в `/Library/Keyboard Layouts/`.
## Наши раскладки
- m4de qwerty → `org.unknown.keylayout.m4deqwerty`
- RussianWin → `com.apple.keylayout.RussianWin`
- Australian → системная, лишняя
## Решение
Создать файл `~/.config/karabiner/assets/complex_modifications/toggle_two_layouts.json`.
Команда для xonsh:
```xonsh
@('''{
  "title": "Toggle between two layouts",
  "rules": [
    {
      "description": "Caps Lock toggles between m4de qwerty and RussianWin",
      "manipulators": [
        {
          "type": "basic",
          "from": { "key_code": "caps_lock" },
          "to": [{ "select_input_source": { "input_source_id": "com.apple.keylayout.RussianWin" } }],
          "conditions": [{ "type": "input_source_if", "input_sources": [{ "input_source_id": "org.unknown.keylayout.m4deqwerty" }] }]
        },
        {
          "type": "basic",
          "from": { "key_code": "caps_lock" },
          "to": [{ "select_input_source": { "input_source_id": "org.unknown.keylayout.m4deqwerty" } }],
          "conditions": [{ "type": "input_source_if", "input_sources": [{ "input_source_id": "com.apple.keylayout.RussianWin" }] }]
        },
        {
          "type": "basic",
          "from": { "key_code": "caps_lock" },
          "to": [{ "select_input_source": { "input_source_id": "org.unknown.keylayout.m4deqwerty" } }]
        }
      ]
    }
  ]
}''') > ~/.config/karabiner/assets/complex_modifications/toggle_two_layouts.json
```

Третий manipulator — fallback: если активна системная Australian, переключает на m4de qwerty.
## Активация
Karabiner-Elements → Complex Modifications → Add predefined rule → "Toggle between two layouts" → Enable.
