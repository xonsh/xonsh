Overview
========================

Xonsh is a Python-based shell that requires Python to be packaged, preinstalled or compiled in order to run.
Since there are many ways to install Python, there are also many ways to run xonsh. This table describes
the main approaches and their advantages. Below, we will look at each of them in more detail and provide
installation instructions.
