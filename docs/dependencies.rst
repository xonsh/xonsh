Dependencies
============
Xonsh currently has the following external dependencies,

*Run Time:*

    #. Python v3.4+
    #. prompt-toolkit (optional)
    #. Jupyter (optional)
    #. setproctitle (optional)

*Documentation:*

    #. `Sphinx <http://sphinx-doc.org/>`_ (which uses  `reStructuredText <http://sphinx-doc.org/rest.html>`_)
    #. Numpydoc
    #. Cloud Sphinx Theme
